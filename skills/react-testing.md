name: react-testing
description: React component testing with Testing Library, Vitest, Cypress
instructions: |
  Test React components following these principles:
  1. Test behavior, not implementation
  2. Use @testing-library/react for unit tests
  3. Use Vitest for test runner
  4. Use Cypress or Playwright for E2E
  5. Test: rendering, user interactions, error states, loading states
  6. Mock external dependencies at the boundary
  Prefer user-centric queries (getByRole, getByText) over test IDs.
